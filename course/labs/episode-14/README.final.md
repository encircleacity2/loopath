# Loopath

Loopath is a mini Codex-style coding agent runtime built from scratch.

It demonstrates how to turn model outputs into safe, observable, and evaluable
actions:

- structured action schema
- tool registry
- policy checks
- context strategies
- trace logging
- eval runner
- self-repair
- loop engineering experiments
