# Episode 1: Loopath Boundaries

## Learning Objectives

- Distinguish model, agent, harness, and world.
- Explain why a harness is not an API wrapper.
- Create the first Loopath project skeleton.
- Understand README, AGENTS.md, and architecture.md.
- Use a verifier to check whether Lab 1 is complete.

## Step Map

1. What Loopath is
2. What you will build
3. Why a harness is not an API wrapper
4. Model / Agent / Harness / World boundaries
5. Lab 1 target
6. Create the directory structure
7. README as the product promise
8. Architecture as the core constraints
9. Draw the first core loop
10. AGENTS.md as the scenario protocol
11. pyproject.toml as the Python project configuration
12. Verify Lab 1
13. Ask an agent to grade by rubric
14. Quiz one question at a time

## Teaching Style

Show one small topic at a time. After each topic, offer:

- continue to the next step
- ask a question about the current step
- start the lab
- run verification
- start the quiz
