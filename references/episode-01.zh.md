# Episode 1：Loopath 的边界

## 学习目标

- 区分 model、agent、harness、world。
- 解释为什么 harness 不是 API wrapper。
- 创建 Loopath 的第一版项目骨架。
- 理解 README、AGENTS.md、architecture.md 的作用。
- 用 verifier 检查 Lab 1 是否达标。

## Step Map

1. Loopath 是什么
2. 你最终会构建什么
3. 为什么 Harness 不是 API Wrapper
4. Model / Agent / Harness / World 四层边界
5. Lab 1 的目标
6. 创建目录结构
7. README 是产品承诺
8. Architecture 写核心约束
9. 画出第一版 core loop
10. AGENTS.md 是场景工作协议
11. pyproject.toml 是 Python 项目配置入口
12. 用 verifier 验收 Lab 1
13. 让 agent 按 rubric 检查
14. Quiz 一问一答

## Teaching Style

每次只讲一个小课题。讲完后，给用户选择：

- 继续下一节
- 针对当前小节提问
- 开始 Lab
- 跑 verification
- 开始 quiz
